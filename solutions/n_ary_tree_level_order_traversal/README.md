# N-ary Tree Level Order Traversal

Дано N-нарное дерево.

Верните порядок обхода его значений узлов.

### Пример 1

            1
          / | \ 
         /  |  \
        3   2   4
       / \
      5   6

Результат должен быть

```python
result = [[1],[3,2,4],[5,6]]
```
