# ReverseLinkedList Problem

## Описание
На вход подается ссылка на голову связанного списка. Необходимо развернуть его и вернуть ссылку на новую голову, которая является последней нодой изначального связанного списка

### Как это выглядит ?
```
ReqL: 1  ->  2  ->  3  ->  4  ->  5

                    |
                    v

ResL: 5  ->  4  ->  3  ->  2  ->  1
```
