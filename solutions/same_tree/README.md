# Same Tree

## Описание:

Имея корни двух бинарных деревьев p и q, напишите функцию, проверяющую, совпадают ли они или нет.

Два бинарных дерева считаются одинаковыми, если они структурно идентичны, а узлы имеют одинаковое значение.

## Пример

```
    1       1
   / \     / \ 
  2   3   2   3

result = True
```

## Пример

```

    1       1
     \     /  
      3   3  

result = False
```
