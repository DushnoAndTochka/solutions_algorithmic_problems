# Is power of 2?

## Описание:
На вход мы получаем число n

Нужно вернуть является ли это число степенью двойки?

Ограничения:

```python
-2^31 <= n <= 2^31 - 1
```


А можно ли решить эту задачу без циклов?

---

### Пример 1:

```python 
n = 4

answer = True
```

---

### Пример 2:

```python 
n = -4
answer = False
```

--- 

### Пример 3:

```python 
n = 0
answer = False
```
