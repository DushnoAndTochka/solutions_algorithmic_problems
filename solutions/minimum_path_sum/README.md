# 64. Minimum Path Sum


Дана сетка m x n, заполненную неотрицательными числами, 
найдите путь из верхнего левого угла в нижний правый, 
который минимизирует сумму всех чисел на своем пути.


## Пример

```python

grid = [
    [1,3,1],
    [1,5,1],
    [4,2,1]
]

result = 7 # 1 -> 3 -> 1 -> 1 -> 1
```
