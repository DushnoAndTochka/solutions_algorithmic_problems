# Subsets

## Описание:

Дан массив уникальных чисел.

Нужно вернуть все возможные подмножества

## Пример

```python
nums = [1,2,3]
result = [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```

## Пример

```python
nums = [0]
[[],[0]]
```
