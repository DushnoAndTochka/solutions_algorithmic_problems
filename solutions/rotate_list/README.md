# Rotate List Problem

## Описание:

На вход подается ссылка на голову связанного списка и число `k`. Необходимо сдвинуть связанный список `k` раз.

## Пример

```
head
  |
  V
  1 -> 2 -> 3 -> 4 -> 5

k = 3

# first rotate
head
  |
  V
  5 -> 1 -> 2 -> 3 -> 4


# second rotate
head
  |
  V
  4 -> 5 -> 1 -> 2 -> 3
```
