# TwoSum Problem

## Описание:
На вход подается **множество**, которое состоит из чисел и **таргет**. Необходимо найти индексы двух чисел, сумма которых дает таргет.

## Пример:
```python 
nums = [1,2,3,4,5]
target = 6

answer = [0, 2]
```

```python 
nums = [15,-2,23,-4,65]
target = 63

answer = [1, 4]
```
