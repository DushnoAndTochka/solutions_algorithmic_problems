# Single Element in a Sorted Array Problem

## Описание

На вход дается отсортированный массив целых чисел. Каждое чило, кроме одного, встречается 2 раза. Необходимо найти это число и вернуть.

Пример:

```
nums = [1, 1, 2, 3, 3, 4, 4]

result = 2
```
