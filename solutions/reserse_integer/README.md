# Reverse Integer

На вход получаем 32-х битное число.

Нужно вернуть число с перевернутыми числами.

Если результат превышает максимальное значение 32-х битного числа то нужно вернуть 0.


### Пример 1.

```python

x = 123

result = 321
```


### Пример 2.

```python

x = -123

result = -321
```


### Пример 3

```python
x = 120

result = 21
```
